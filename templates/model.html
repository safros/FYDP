Description of the Model:

This model is based on an mTSP. The method used to solve it is heuristics.

The construction heuristic is based on a nearest neighbor approach. The improvement heuristic is a 2-opt switch.

The data is first put into a graph and the shortest path algorithm is run for each of the demand nodes and start location.

The shortest path creates an adjacency matrix. The matrix is used in the heuristic.

The model uses data collected from ARCGIS. The demand of the nodes is generated randomly. The demand nodes are chosen randomly.

The capacity of the trucks is generated.
