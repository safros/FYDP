How to format the CSV

There should be 8 tabs:

  • • distance
  • • truck
  • • demand
  • • start
  • • damages
  • • speedLimit
  • • emissions
  • • nodeLookUp
  1. The distance tab has the columns: Origin_ID, Destination_ID, and Length_m. This tab is the distances for each node.
  2. The truck tab has the columns: id_truck, capacity, type, gasVdiesel, and esal. This tab is the information about the truck.
  3. The demand tab has the columns: node_id and demand_units. This tab is all the information abotu the demand.
  4. The start tab has the column: clcLocation. It holds the start location of the trucks.
  5. The damages tab has the columns: Origin_ID, Destination_ID, and Trip_Cost_dollar. This tab has the information about the damages based on the road classifications.
  6. The speedLimit tab has the columns: Origin_ID, Destination_ID, and Speed_Limit. This tab holds all the speeds for each arc. This is used for emissions calculations.
  7. The emissions tab has the columns: typeTruck,gasVdiesel, speed, costperKm. This tab holds all the simulations of emissions for different types of trucks.
  8. The nodeLookUp tab has the columns: lookup_ID, Latitude, and Longitude. It holds the mapping information for all the nodes in the graph.
